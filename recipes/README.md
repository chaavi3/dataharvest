# Recipes

Step-by-step guides for specific data extraction use cases using DataHarvest.

Each recipe walks through a real-world scenario end-to-end -- from choosing the right source type and configuring columns to running the job and exporting results.

## Available recipes

| Recipe | Description |
|---|---|
| [IPL MVP Stats](ipl-mvp-stats.md) | Scrape player-level MVP / Total Impact data from every IPL 2025 match on ESPNcricinfo |

## Contributing a recipe

Have an interesting use case? Add it here! To contribute:

1. Create a new markdown file in this folder (e.g., `your-use-case.md`)
2. Follow the general structure: what the goal is, which source type to use, column definitions, step-by-step instructions, and any tips
3. Add your recipe to the table above
4. Open a pull request

Recipes don't need to be complicated -- even a short walkthrough that saves someone an hour of figuring things out is valuable.
